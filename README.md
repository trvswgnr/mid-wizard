### Getting Started:
Clone this repo & enter
```bash
git clone https://github.com/trvswgnr/mid-wizard && cd mid-wizard
```
---
Install dependencies
```bash
npm install
```
---
Start task runner (Gulp), local server, and watch for changes.
```bash
gulp
```
---
Production build:
```bash
gulp clean && gulp --production
```
